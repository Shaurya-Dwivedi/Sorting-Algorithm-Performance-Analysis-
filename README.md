# Sorting Algorithm Performance Analysis | R, Statistical Modeling | MAL2040, IIT Jodhpur

- Designed a 3² factorial experiment (45 runs, 5 machines as blocks) to benchmark Quick Sort, Heap Sort, and Merge Sort; confirmed Input Size as the dominant runtime driver (p = 1.64×10⁻⁸) via ANOVA
- Diagnosed ANOVA assumption violations using Bartlett's test and residual plots; applied RCBD blocking and Response Surface Methodology to confirm O(n log n) empirical scaling behavior
